html 문서는 크게 head 영역과 body 영역으로 나누어 진다.

body -> 브라우저 화면에 보여지는 영역

head -> 화면에 보여지지는 않는다. html 문서에 대해서 알아야 할 정보를 가지고 있는다.

자주 쓰이는 태그

title 태그

브라우저 상단에 제목을 정의하기 위해 사용된다.

반드시 하나만 와야한다 -> 에러가 발생하지는 않는다. 처음으로 정의 된 제목을 쓴다.

link 태그

외부에 위치하고 있는 리소스를 불러올 때 사용한다.

rel = 현재 웹페이지와 외부 리소스의 관계를 명시한다.

href = 외부 리소스의 경로를 명시한다.

주로 CSS파일이나 스크립트 파일을 불러올 때 사용한다.

favicon을 불러올 때도 사용한다.

style 태그

외부 CSS 파일을 불러올 수도 있지만 html 문서 내부에 CSS 코드를 작성해서 스타일을 적용할 때 쓴다.

script 태그

웹페이지에서 필요한 자바스크립트 코드를 추가하기 위해서 사용한다.

src 속성을 이용해서 자바스크립트 파일 경로를 지정할 수 있다.

CSS처럼 html 문서 내부에 작성할 수 있다.

base 태그

웹페이지에서 사용된 주소의 기본 url 주소와 기본 타겟을 지정한다. 맑은소프트 LMS

meta 태그

메타데이터란 데이터를 설명하는 데이터이다.

웹페이지 자체의 정보를 명시하기 위한 목적으로 사용한다.

charset -> 웹페이지의 문자 인코딩 방식을 명시한다.

http-equiv -> HTTP 헤더를 HTML 문서 내부에서 명시하기 위해서 사용됩니다.

content-security-policy = 문서에 대한 콘텐츠 보안정책을 지정한다.

content-type = 문서의 문자 인코딩을 지정한다.

default-style = 사용할 선호 스타일 시트를 지정한다.

refresh = 문서가 자동으로 새로 고쳐지는 시간(초 단위)을 지정한다.
